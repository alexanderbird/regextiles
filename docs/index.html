Regex Tiles Level

Eliminate the falling tiles in real-time with your sweet RegEx skills.

Gameplay

  • Type some RegEx to match tiles
  • You're matching against the bottom row
  • Empty columns are treated as spaces
  • You're only allowed to match tiles of one color at a time
  • If you match more than one color, your match is invalid
  • Your regex is applied when the timer expires or when you press Enter
  • The longer you wait before pressing Enter, the more tiles that will be released
  • Your regex is applied with the "g" (global) flag so it can match multiple parts of the line

Tips

Matching Tiles

  • Use . to match any character
  • You can match exactly one letter by typing only that letter
  • You can match multiple letters by joining them with pipe characters (|)
  • You can match everything between two letters with X.*Y (which means "X followed by anything followed by Y")

Avoiding Certain Tiles

  • Use spaces to pinpoint a specific letter (e.g. use A to match an "A" that has no tile before it)
  • You can match a sequence of letters by typing them all out
  • You can match "any character except x or y" with the regex [^xy]
  • You can match characters relative to the start or end with ^ and $

Repeating Characters

  • Use + to repeat a character one or more times (e.g. match aaaaaa with a+
  • Use * to repeat a character zero or more times (e.g. match aaaaaa or the empty string with a+
  • Use {n} to repeat a character n times (e.g. match aaaaaa with a{6}

Character Classes

  • [xyz] defines a character class, which means "one tile that contains either x or y or z"
  • Use a range within a character class (e.g. match abcd with [a-d])
  • Common character class ranges: A-Z, a-z, 0-9
  • A character class can have several ranges (e.g. match any alphanumeric with [A-Za-z0-9]
  • Use ^ inside square braces to negate a character class to match everything except (e.g. match efgh with [^a-d])

Special Characters

  • To match a character that usually has a special meaning, you will need to escape that character with backslash
  • Use \X to escape some character X (e.g. match $ with \$)
